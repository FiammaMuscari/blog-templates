### Sintaxis de Redux

```js

// Reducer

  switch (action.type) {
    case 'INCREMENT':
      return state + 1
    case 'DECREMENT':
      return state - 1
    default:
      return state
  }

// Acciones

const decrementAction = { type: 'DECREMENT' }
```
